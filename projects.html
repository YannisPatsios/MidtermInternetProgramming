Research Projects

5G-NR ISAC Research

SWiFT Lab, Deree – The American College of Greece

Summer 2024 – Present

Project Overview

Contributing to the development of a 5G-NR ISAC (Integrated Sensing and Communication) demonstration for the EuCNC & 6G summit 2025. This project focuses on leveraging 5G networks for both communication and sensing applications.

Key Responsibilities

  • Managing software-defined radios (SDR) for wireless communication experiments using MATLAB
  • Configuring Raspberry Pi 5 with Armbian OS for Over-The-Air (OTA) channel sounding
  • Simulating 5G Core and Radio Access Network (RAN) for research applications
  • Researching ISAC techniques for analyzing radio signals to measure distance and angle

Technologies Used

MATLAB SDR Raspberry Pi Armbian OS 5G ISAC
